# OHIA FHIR R6 Scratchpad
Working repository for drafting FHIR R4 API definitions.

## Project Details
- **FHIR Version:** R4 (4.0.1)
- **Primary Format:** JSON
- **Status:** Draft / Work-in-Progress

## Folder Guide
- /profiles: Custom StructureDefinitions (JSON)
- /examples: Sample resource instances (JSON)
- /terminology: ValueSets and CodeSystems
- /extensions: Custom FHIR extensions
